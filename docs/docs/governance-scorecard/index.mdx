# Governance Compliance Scorecard

:::warning[Alpha Research — AI-Generated Analysis]
This scorecard is an **alpha-stage research prototype**. The scores, grades,
and alerts were generated using AI analysis of legislative texts with human
advisory input. **The data and model have not been independently verified.**

- Do **not** use these scores to make policy, legal, or administrative conclusions
- The Lanka Data Foundation (LDF) accepts **no responsibility** for any
  decisions or actions taken based on this analysis
- Methodology and scoring parameters are subject to revision
- Source data confidence varies (see per-act confidence ratings)

This tool is intended solely for **research and exploratory purposes** to
demonstrate how legislative mandate compliance could be systematically measured.
:::
