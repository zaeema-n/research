# Act Deep-Dive — Agent Knowledge Base

This directory captures the complete knowledge an AI agent needs to efficiently research, model, and publish a legislative act deep dive. Built from experience across 20+ act deep dives (Health Ministry: 12 acts, Education Ministry: 7 acts/ordinances).

## Quick Start for New Sessions

1. **Read `context.md`** — current project state, what's done, what's next
2. **Read `ui-update-checklist.md`** — the exact 11-file checklist per act
3. **Read `extraction-workflow.md`** — how to research and extract act data
4. **Consult `lessons-learned.md`** — known pitfalls and fixes

## Files in This Directory

| File | Purpose |
|------|---------|
| `extraction-workflow.md` | Research methodology: how to find, read, and extract act information from source documents |
| `ui-update-checklist.md` | Per-act file creation/modification checklist for both Docusaurus and React app |
| `data-model-reference.md` | JSON schemas, OpenGIN entity model, naming conventions |
| `context.md` | Current state snapshot for session resumption |
| `lessons-learned.md` | Accumulated pitfalls, fixes, and patterns |
